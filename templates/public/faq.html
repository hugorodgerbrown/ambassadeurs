{% comment %} Section: Eligibility {% endcomment %}

{% translate "Eligibility" %}

{% translate "Who qualifies as an Ambassador?" %}

{% blocktranslate trimmed %} An Ambassador is someone who held a 4 Vallées seasonal or annual pass — or a Mont 4 Card / special-reduction pass — in one of the previous two seasons. When you register you self-attest to this; proof is checked off-app during the joint application process with the pass operator. {% endblocktranslate %}

{% translate "Who qualifies as a Referee?" %}

{% blocktranslate trimmed %} A Referee is someone who did not hold any 4 Vallées season pass in either of the two previous seasons. This is self-attested on registration and verified at the ticket office when you apply together. {% endblocktranslate %}

{% translate "Can a Mont 4 Card holder act as an Ambassador?" %}

{% blocktranslate trimmed %} Yes. A Mont 4 Card or special-reduction pass counts as a qualifying prior pass. The Referee they sponsor still benefits from the referral discount even if the Ambassador does not receive one themselves. {% endblocktranslate %}

{% comment %} Section: How matching works {% endcomment %}

{% translate "How matching works" %}

{% translate "Do I choose my own partner?" %}

{% blocktranslate trimmed %} No — the system assigns a match. You register once, and when an eligible partner is available the system proposes a pairing. You do not browse profiles or choose a partner. {% endblocktranslate %}

{% translate "How does the system decide who to pair me with?" %}

{% blocktranslate trimmed %} The service pairs eligible participants in order of registration date (earliest first). Both parties must be verified with no active match already in progress. {% endblocktranslate %}

{% translate "How long until I get a match?" %}

{% blocktranslate trimmed %} If an eligible partner is already waiting when you register, a match is proposed immediately. Ambassadors typically outnumber Referees, so Referees are often matched instantly; Ambassadors may wait in the queue. You can see your position in the queue from your account page. {% endblocktranslate %}

{% comment %} Section: The contact window {% endcomment %}

{% translate "The contact window" %}

{% translate "What is the contact window?" %}

{% blocktranslate trimmed %} Once a match is proposed, both parties have a limited window — its length is shown in the match email — to accept. Neither side can see the other's contact details (email or phone) during this window; only your partner's first name is shown so the pairing feels human. Contact details are revealed only after both parties accept. {% endblocktranslate %}

{% translate "Why can't I see my partner's contact details before accepting?" %}

{% blocktranslate trimmed %} Contact details (email and phone) are hidden until both parties have accepted the match. This protects everyone's privacy: if either side declines or the window expires, neither party's contact information is ever shared. {% endblocktranslate %}

{% translate "What happens if I miss the contact window?" %}

{% blocktranslate trimmed %} If you do not respond within the window, the match expires and your registration is paused. You can rejoin the queue from your account page whenever you are ready. Your partner, if they accepted the match, returns to the front of the queue. {% endblocktranslate %}

{% translate "What happens if my partner misses the contact window?" %}

{% blocktranslate trimmed %} If your partner does not respond in time, the match expires and you are moved to the front of the queue. The system will propose a new match for you as soon as an eligible partner is available. {% endblocktranslate %}

{% translate "How long is the contact window?" %}

{% blocktranslate trimmed %} It starts at 72 hours — long enough to cover a weekend away from email — and shortens towards 24 hours, possibly less, as the season start approaches and things get busier. The current window is always shown in the match email. {% endblocktranslate %}

{% comment %} Section: After matching {% endcomment %}

{% translate "After matching" %}

{% translate "What happens once both of us have accepted?" %}

{% blocktranslate trimmed %} Once both parties accept, each receives the other's contact details. You must then arrange to fill in the application form together and submit it to the ticket office. The pass purchase and application happen off this site, unchanged from the normal process. {% endblocktranslate %}

{% translate "What if my partner doesn't show up after we've both accepted?" %}

{% blocktranslate trimmed %} If your partner fails to follow through after mutual acceptance you can report them as a no-show. The report is acted on immediately: you re-enter the queue at the front, and your partner's registration is suspended. {% endblocktranslate %}

{% translate "Does this site handle the application form or pass purchase?" %}

{% blocktranslate trimmed %} No. This site handles the matchmaking only — connecting an Ambassador with a Referee so they can apply together. Filling in the application form and purchasing the passes at the ticket office are unchanged and happen entirely off this site. {% endblocktranslate %}

{% comment %} Section: Applying {% endcomment %}

{% translate "Applying" %}

{% translate "What is the 4 Vallées Ambassadors Programme?" %}

{% blocktranslate trimmed %} The 4 Vallées Ambassadors Programme is a referral scheme that lets a returning season-pass holder (an Ambassador) bring a new person (a Referee) into the programme. Both apply together and, if eligible, both receive a discount on their seasonal pass. This site handles the matchmaking — it finds an Ambassador a Referee, and vice versa — replacing the chaotic scramble for a partner that used to happen in community groups. {% endblocktranslate %}

{% translate "How do I apply?" %}

{% blocktranslate trimmed with email="customer@televerbier.ch" %} Download the application form, fill it in together with your partner, and email it to {{ email }} (Téléverbier). Both parties must sign the form and provide proof of identity (passport or national ID card) and a digital passport photo. If applying by email these can be photograph attachments. {% endblocktranslate %}

{% translate "Download application form" %}

{% translate "What is the approval process?" %}

{% blocktranslate trimmed %} Once Téléverbier receives the application form they will assess your eligibility. If approved, they will issue a payment link or invoice. Once payment is received your passes will be approved. You can then collect your physical pass cards at the ticket desk, or have them loaded onto your existing keycards if you provided your keycard numbers in the application. {% endblocktranslate %}

{% translate "What are the requirements?" %}

{% blocktranslate trimmed %} Both the Ambassador and the Referee must sign the application form and provide a valid passport or national identity card. Both must attest to the fact that they have or have not had a qualifying pass (as appropriate); this will be checked during the application process. {% endblocktranslate %}
