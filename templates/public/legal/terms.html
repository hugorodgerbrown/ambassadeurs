{% translate "What this service does" %}

{% blocktranslate trimmed %} This service exists to help you find a partner for the 4 Vallées Ambassadors Programme. It is matchmaking only: the ski pass application process (and payment) are done directly by you with the pass operator. {% endblocktranslate %}

{% translate "What this service costs" %}

{% blocktranslate trimmed %} This service is provided free of charge. If you get value from the service, donations are welcome. {% endblocktranslate %}

{% translate "How it works" %}

  • Users register on the site as either Ambassador or Referee (as appropriate).
  • Registered users verify their email by clicking on a link supplied to that email address.
  • Verified users are then placed in a matching queue, which operates on a First in, first out basis.
  • As soon as someone is available, a match is proposed, and both parties notified.
  • If both parties accept the match, contact details are shared through the site, and the process is complete.
  • {% blocktranslate trimmed %} If a user declines a match, their registration is paused — they may rejoin the queue from their account page. {% endblocktranslate %}
  • {% blocktranslate trimmed %} If a user does not respond in time, their registration is paused — they may rejoin the queue from their account page. {% endblocktranslate %}
  • If one user accepts the match, but the other user declines or fails to respond, they will return to the front of the queue.

{% translate "Your responsibilities" %}

  • {% translate "Give accurate information, including your honest prior-season eligibility." %}
  • {% translate "Respond to a match within the window shown in the match email — accept or decline." %}
  • {% blocktranslate trimmed %} If you do not respond in time, your registration is paused — you can rejoin the queue from your account page when you are ready. {% endblocktranslate %}
  • {% translate "Be respectful and prompt with the partner you are matched with." %}

{% translate "Eligibility" %}

{% blocktranslate trimmed %} Eligibility is self-declared here, and is used for matchmaking only. Your eligibility will be verified separately off-app when you apply for the season pass. Registering here does not guarantee a match or entitle you to the discount; both depend on the application process with the operator. {% endblocktranslate %}

{% translate "No guarantee" %}

{% blocktranslate trimmed %} This is a two-sided marketplace, there will always be more people on one side than the other - hence no guarantee of a match is possible. {% endblocktranslate %}
