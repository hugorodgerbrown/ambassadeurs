{% translate "How it works" %}

{% blocktranslate trimmed %} Two things carry a status here, and it helps to keep them apart: your registration, and a match. You always have exactly one registration — your own standing in the system. Over a season you might be part of several matches, since an early one can fall through and you go back in the queue to be paired again. {% endblocktranslate %}

{% translate "Part 1 — Your registration" %}

{% blocktranslate trimmed %} Your registration is your place in the system — the answer to "am I in the queue, waiting to be matched?" {% endblocktranslate %}

{% translate "Status" %} {% translate "What it means" %} {% translate "What happens next" %}
{% translate "Unverified" %} {% blocktranslate trimmed %} You've signed up but haven't confirmed your email address yet. You are not in the queue. {% endblocktranslate %} {% blocktranslate trimmed %} Click the link in the confirmation email. That moves you to Verified. {% endblocktranslate %}
{% translate "Verified" %} {% blocktranslate trimmed %} You're active and in the queue, waiting to be matched (or already in an active match). {% endblocktranslate %} {% blocktranslate trimmed %} The system pairs you with an eligible partner as soon as one is available. When it does, a match is created. {% endblocktranslate %}
{% translate "Paused" %} {% blocktranslate trimmed %} You declined a match, or didn't respond in time, so you've been taken out of the queue. Nothing is lost — your registration is kept. {% endblocktranslate %} {% blocktranslate trimmed %} Go to your account page and choose "Rejoin the queue". That moves you back to Verified and you'll be matched again. {% endblocktranslate %}
{% translate "Withdrawn" %} {% translate "You chose to leave the programme." %} {% blocktranslate trimmed %} Nothing further. You can register again if you change your mind. {% endblocktranslate %}
{% translate "Suspended" %} {% blocktranslate trimmed %} You were reported as a no-show after a match was confirmed, so you've been removed from the pool. {% endblocktranslate %} {% blocktranslate trimmed %} This is set by programme staff via the report process, not something you undo yourself. Contact the programme if you think it's a mistake. {% endblocktranslate %}

{% blocktranslate trimmed %} In short: Verified is the healthy, waiting state. Everything else is either a step before it (Unverified) or a step out of the pool (Paused, Withdrawn, Suspended). Only Paused has a self-service way back in. {% endblocktranslate %}

{% translate "Part 2 — A match" %}

{% blocktranslate trimmed %} Once you're Verified and a suitable partner is available, the system creates a match — a pairing of one Ambassador and one Referee. You don't choose your partner; the system assigns one, and both of you are notified. From that moment there is a contact window — its length is shown in the match email — in which both of you must accept. Until you both accept, you can each see the other person's first name, but not their email or phone number: contact details are revealed only once both sides have accepted. This is deliberate — it's the core privacy promise of the service. {% endblocktranslate %}

{% translate "Status" %} {% translate "What it means" %} {% translate "What happens next" %}
{% translate "Proposed" %} {% blocktranslate trimmed %} You've just been paired. Neither of you has accepted yet. You can see your partner's first name only. {% endblocktranslate %} {% blocktranslate trimmed %} Accept within the contact window. If your partner accepts first, you'll see the match move to Pending. {% endblocktranslate %}
{% translate "Pending" %} {% blocktranslate trimmed %} One of you has accepted; the system is waiting on the other. {% endblocktranslate %} {% blocktranslate trimmed %} The second person accepts and the match becomes Accepted. If the window runs out with only one acceptance, it becomes Expired. {% endblocktranslate %}
{% translate "Accepted" %} {% blocktranslate trimmed %} Both of you accepted. Contact details are now revealed. This is the success state. {% endblocktranslate %} {% blocktranslate trimmed %} Get in touch with your partner and go make your application together — this happens off the app. {% endblocktranslate %}
{% translate "Declined" %} {% translate "One of you declined the pairing." %} {% blocktranslate trimmed %} The match is over. The person who declined is Paused; the other person goes back to the front of the queue to be matched again quickly. {% endblocktranslate %}
{% translate "Expired" %} {% blocktranslate trimmed %} The contact window ran out before both of you accepted. {% endblocktranslate %} {% blocktranslate trimmed %} Same as declined: whoever didn't respond is Paused; the one who did accept goes to the front of the queue. {% endblocktranslate %}
{% translate "Cancelled" %} {% blocktranslate trimmed %} After a match was accepted, one party reported the other as a no-show. {% endblocktranslate %} {% blocktranslate trimmed %} The person who showed up goes back to the front of the queue; the reported person is Suspended. {% endblocktranslate %}

{% blocktranslate trimmed %} In short: Accepted is the goal. Declined, Expired and Cancelled all end the match — but they don't end you: unless you were the one who dropped out, you go back to the front of the queue and are matched again. {% endblocktranslate %}
